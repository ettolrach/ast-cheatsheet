\wde{n.n.} For section 2.3, we define $\sim$ for the relation on $\N \times \N$ defined by $(m, n) \sim (m', n') \iff m + n' = m' + n$ ($m, m', n, n' \in \N$). (Lemma 7.3.1) $\sim$ is an equiv. rel.
\wde{7.3.2 $\Z$} $\Z$ is $(\N \times \N) / \sim$.
\wl{7.3.5} Let $\sim_1$ be an equiv. rel. on a set $X$ and $\sim_2$ an equiv. rel. on a set $Y$. Then the relation $\sim$ on $X \times Y$ defined by $(x, y) \cong (x', y') \iff x \sim_1 x' \text{ and } y \sim_2 y'$ ($x, x' \in X, y, y' \in Y$) is an equiv. rel. Moreover, $(X \times Y) / \sim \cong X / \sim_1 \times Y / \cong_2$.
\wpr{7.3.9} $\Z$ is a commutative ring. (lemma 7.3.12) $\leq$ a total order, (lemma 7.3.13) $\Z$ an ordered ring.
