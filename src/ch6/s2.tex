\wde{no number, Equivalence class} Let $E$ be an equivalence relation on a set $X$. The \textit{equivalence class} of an element $x \in X$ is the subset $E[x]$ of $X$, as defined in lemma 5.1.11. If we write the equiv. rel. as $\sim$, then we denote the equivalence class of $x$ by $[x]_{\sim}$ or just $[x]$.
\wl{6.2.3} Let $\sim$ be an equivalence relation on a set $X$. Then for $x, y, \in X$, $x \sim y \iff [x] = [y]$.
\wde{no number, Quotient} Let $E$ be an equiv. rel. on a set $X$. Then $E$ corresponds to a function $\bar{\epsilon} X \to \Pow(x)$, $x \mapsto E[x]$. Define $X/E$ the \textit{quotient of $X$ by $E$} to be the image of $\bar{\epsilon}$.

\wlog{6.2.5} Let $\sim$ be an equivalence relation on a set $X$. Then for $x, y \in X$, $\pi_{\sim}(x) = \pi_{\sim}(y) \iff x \sim y$.
